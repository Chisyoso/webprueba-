# timbitoOS v4

## GitHub Pages
Sube `index.html`, `css/styles.css`, `js/app.js` y `esp32_sketch.ino`.

## ESP32
1. Cambia `TU_WIFI` y `TU_PASS`.
2. Sube el sketch al ESP32.
3. Abre la web y agrega la dirección del ESP32.

## Nota
Si el navegador bloquea `http://` desde una página `https://`, abre el proyecto en local o usa un proxy/HTTPS para el ESP32.
